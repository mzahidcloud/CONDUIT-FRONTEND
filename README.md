**Conduit** is a social blogging application (a Medium.com-style clone) built with Angular. It demonstrates a realistic single-page app with user authentication, full CRUD for articles and comments, tags, favorites, following other users, and paginated article feeds.

## Practice project for Bondar Academy

This repository is used as a hands-on application for learning and practicing **test automation** at [Bondar Academy](https://www.bondaracademy.com/). Students use it as a realistic, full-featured web app to write and run automated tests against.

Explore the available training programs at **https://www.bondaracademy.com/**.

## Running the app

Requires Node.js `^18.13.0 || ^20.9.0`.

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the dev server:

   ```bash
   npm start
   ```

3. Open http://localhost:4200/ in your browser. The app reloads automatically when you change source files.

## Building

Run `npm run build` to build the project. The build artifacts are output to the `dist/` directory.
